# Alist文件列表

## 是什么？
`V3 一个支持多种存储，支持网页浏览和 WebDAV 的文件列表程序，由 gin 和 Solidjs 驱动。`

`V2 一款支持多种存储的目录文件列表程序，支持 web 浏览与 webdav，后端基于gin，前端使用react。`


## 项目地址
https://github.com/alist-org/alist
